# Localboards
Todo list app that stores data locally

## Demo :gem:
[localboards.ggez.se](https://localboards.ggez.se)

## Features :clipboard:
- Stores all data in localstorage
- Download/Upload option
- Light/Dark mode

## Stack :zap:
- HTML, CSS, TS
- [SoloJSX](https://github.com/SoloJSX/SoloJSX)
- Vite
